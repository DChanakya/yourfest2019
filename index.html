Code War

CODE WAR

Code War

Event Co-ordinator: Mr T.Narasimha Appadu, CSE , Ph:9492638410

Round 1:

screening test:-

In this round, 20 bits will be given to each participant and the time limit is 20minutes.

The questions consists of:

Basics of C language

Simple logic evaluation bits.

 

Rules for round1:

Participants should complete the online exam within 20 minutes

The qualified students will be promoted to the second round.

 

Round 2:

Implementation of logic:-

In this round, a logic/problem statement will be given and the students need to generate their own code with accurate output.

The students will be provided with a system and the expected output should be generated.

 

Rules for round2:

Logic is common to all the participants.

Time limit is 30 minutes

The students selected in the second round will be promoted to the third round.

 

Round 3:

Debugging:-

In this, the participant should detect the errors in the given program and display the desired output.

 

Rules for round3:

In this round, the participants should detect the error within the given time.

If more than one participants detect the error at the same time, their marks in the first two rounds will be considered.

The time limit for the third round will be announced at the time of event.
